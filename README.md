Memey is a webapp for browsing memes scraped from popular sites like reddit and imgur.

Gallery: Browse through memes in a scrolling gallery
Twitter Integration: View embedded tweets
Responsive Design: Optimized for desktop (mobile users are redirected to a not-supported page)

Technologies Used

Frontend: Angular 15 with TypeScript
Backend: Firebase (for storage and hosting)
UI Framework: Angular Material and Bootstrap 5

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 15.2.5.
